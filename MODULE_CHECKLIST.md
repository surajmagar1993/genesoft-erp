## 🌍 Multi-Country Tax & Compliance

### 🇮🇳 India (P1 — Your Home Market)
- [x] **GST (CGST + SGST + IGST)** 🔴 — Intra-state & inter-state tax
- [x] **HSN / SAC Codes** 🔴 — Product & service classification (e.g., 997315)
- [x] **GSTIN Validation** 🔴 — Verify customer/vendor GST numbers
- [x] **Place of Supply Rules** 🔴 — Auto-determine CGST/SGST vs IGST
- [x] **MSME / Udyam Display** 🔴 — Show Udyam number on invoices
- [x] **TDS (Tax Deducted at Source)** 🟡 — Section 194C, 194J, etc.
- [x] **E-Way Bill Integration** 🟡 — For goods transport > ₹50,000
- [x] **GST Returns Data (GSTR-1, GSTR-3B)** 🟡 — Export-ready reports

- [x] **VAT Return Data** 🟢 — Export for FTA filing
- [x] **Reverse Charge Mechanism** 🟢 — For specific services

- [x] **Zakat Calculation** 🟢 — Islamic wealth tax support
